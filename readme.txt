# LandlordLink

**LandlordLink** is an app designed to connect tenants and building managers, streamlining communication and automating everyday tasks. It helps improve efficiency, reduce response times, and manage requests more effectively for both parties.

## Features
- Connects tenants with building managers in real time.
- Automates routine tasks and maintenance requests.
- Provides a platform for tenants to submit issues or requests.
- Easy-to-use interface for managing rental properties.
- Allow tenants to open their appartments door using phone.
- Track and access package rooms.
- Create and coordinate building events.
